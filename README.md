# Advanced Login System

## Project Overview
This is a Python-based Advanced Login System developed as part of a Cyber Security Internship project. The system allows users to register, log in, and manage user authentication using file handling.

## Features
- User Registration
- User Login Authentication
- Password Strength Checker
- Username Availability Check
- Multiple Login Attempts
- View Registered Users
- File Handling using users.txt
- User-Friendly Menu Interface

## Technologies Used
- Python
- File Handling
- Functions
- Loops
- Conditional Statements
- Regular Expressions (Regex)

## How It Works
1. User selects an option from the menu.
2. New users can register with a username and password.
3. The system checks password strength.
4. User credentials are stored in a text file.
5. Registered users can log in using saved credentials.
6. The system displays success or failure messages.
